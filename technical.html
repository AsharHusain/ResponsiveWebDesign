Introduction

JavaScript is a lightweight, interpreted programming language.

It is used to make web pages interactive and dynamic.

It can update and change both HTML and CSS.

It can be run on the client-side as well as server-side.

JavaScript is one of the core technologies of the web, alongside HTML and CSS.

console.log("Hello, World!"); alert("This is JavaScript!");
  • Client-side language
  • Event-driven
  • Prototype-based
Variables and Data Types

Variables are containers for storing data values.

You can declare variables using var, let, or const.

Data types include String, Number, Boolean, Object, and more.

JavaScript is dynamically typed.

Variables can be reassigned unless declared with const.

let name = "Alice"; const pi = 3.1415;
  • String
  • Number
  • Boolean
  • Null
  • Undefined
Functions and Scope

Functions are blocks of code designed to perform a task.

You can define functions using function declarations or expressions.

Arrow functions offer a shorter syntax.

Scope determines the accessibility of variables.

JavaScript has function and block scopes.

function greet(name) { return "Hello " + name; } const add = (a, b) => a + b;
  • Function declaration
  • Function expression
  • Arrow function
  • Global scope
  • Block scope
DOM Manipulation

The DOM represents the document structure.

You can access elements using methods like getElementById or querySelector.

JavaScript allows dynamic changes to HTML and CSS.

You can create, remove, or edit DOM nodes.

Events are used to interact with user actions.

document.getElementById("title").textContent = "Changed"; element.addEventListener("click", () => alert("Clicked!"));
  • getElementById
  • querySelector
  • createElement
  • appendChild
  • addEventListener
ES6 Features

ES6 introduced many useful features.

Let and Const are preferred for variable declaration.

Arrow functions simplify function syntax.

Template literals allow embedded expressions.

Destructuring simplifies object and array unpacking.

let [a, b] = [1, 2]; const greet = (name) => `Hello, ${name}`;
  • let & const
  • Arrow functions
  • Template literals
  • Destructuring
  • Modules
